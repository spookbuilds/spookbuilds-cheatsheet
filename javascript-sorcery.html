⚡ addEventListener()

JavaScript DOM Events 
                                 
const button = document.querySelector(".button");

button.addEventListener("click", function () {

    alert("Hello!");

});

📜 Notes

  • button → the element you're listening to.
  • addEventListener() → waits for an event.
  • "click" → the event to listen for.
  • "click" can be "input", "keydown" etc.
  • function () → code to run.
  • { } → everything inside runs after the click.

🔗 Often used with...

  • querySelector()
  • querySelectorAll()
  • textContent
  • value
  • preventDefault()

⚡ querySelector()

JavaScript DOM Events 
                                 
const button = document.querySelector(".button");

📜 Notes

  • const→ creates a variable.
  • button→ your variable name.
  • document→ the webpage.
  • querySelector()→ finds an element.
  • ".button"→ looks for a class called button.

⚡ querySelectorAll()

JavaScript DOM Events 
                                 
const cards = document.querySelectorAll(".card");

cards.forEach(card => {

    card.textContent = "Hello";

});

📜 Notes

  • Selects all matching elements.
  • Returns a NodeList.
  • Often used withforEach()
  • Use when multiple elements need changing.

📅 Added: 17 July 2026

⚡ forEach()

JavaScript DOM Events 
                                 
buttons.forEach(button => {

    console.log(button);

});

📜 Notes

  • buttons→ the array or NodeList.
  • forEach()→ loops through every item.
  • button→ current item.
  • =>→ arrow function.
  • {}→ code repeated for every item.
