📖 HTML Document Structure

HTML Beginner Structure

School: HTML Foundations

Every HTML page begins with this structure. It tells the browser what type of document it is and provides the foundation for your webpage.

🕯️ Incantation


    <!DOCTYPE html>

    <html lang="en">

    <head>

        <meta charset="UTF-8">

        <title>
            My Website
        </title>

    </head>

    <body>

    </body>

    </html>

📜 Notes

The head contains information about the page. The body contains everything visible to the user.

✒️ Headings

HTML Beginner Text

School: HTML Text Spells

Headings create hierarchy on a webpage. h1 is the most important heading, while h6 is the smallest. 


    <h1>Main Heading</h1>

    <h2>Section Heading</h2>

    <h3>Sub Heading</h3>

📜 Notes

Use headings in order. Don't choose them based only on size — use them for structure.

✒️ Paragraphs

HTML Beginner Text

School: HTML Text Spells

Paragraph tags create blocks of readable text. They are used for descriptions, explanations, and general page content.

🕯️ Incantation


    <p>
        This is a paragraph of text.
    </p>

📜 Notes

Browsers automatically add spacing between paragraph elements. Use CSS to customise their appearance.

🔗 Link Spells

HTML Beginner Navigation

School: HTML Navigation Magic

Links allow users to travel between pages, websites, and sections of your application.

🕯️ Incantation


    <a href="https://example.com">
        Visit Website
    </a>

📜 Notes

The href attribute contains the destination. Use target="_blank" when opening external links in a new tab.

🖼️ Image Summoning

HTML Beginner Media

School: HTML Media Magic

Images bring visual elements into a webpage. The src attribute tells the browser where to find the image.

🕯️ Incantation


    <img 
    src="image.jpg" 
    alt="Description of image">

📜 Notes

Always include alt text. It improves accessibility and helps users who cannot view the image.

📜 List Rituals

HTML Beginner Organisation

School: HTML Organisation Magic

Lists organise information into readable groups. HTML has ordered and unordered lists.

🕯️ Incantation


    <ul>

        <li>HTML</li>
        <li>CSS</li>
        <li>JavaScript</li>

    </ul>


    <ol>

        <li>Step One</li>
        <li>Step Two</li>

    </ol>

📜 Notes

ul creates bullet lists. ol creates numbered lists. Each item is wrapped in li.

🏛️ Semantic Structure

HTML Intermediate Structure accessibility

School: HTML Architecture

Semantic tags describe the purpose of content. They make websites easier to understand for developers, browsers, and accessibility tools.

🕯️ Incantation


    <header>

    <main>

    <section>

    <article>

    <footer>

📜 Notes

Prefer semantic elements over using divs for everything.

🧙 Forms

HTML Intermediate Forms Input

School: HTML Foundations

Forms are for user input.

🕯️ Incantation


    <form>

    <label>
    Username:
    </label>

    <input 
    type="text"
    name="username">

    <button>
    Submit
    </button>

    </form>

📜 Notes

Forms collect user input. Labels improve accessibility and inputs define what information is collected.

🧱 Div Containers

HTML Beginner Structure Layout

School: HTML Foundations

Container for cards.

🕯️ Incantation


    <div class="card">

    <h2>
    Title
    </h2>

    <p>
    Content
    </p>

    </div>

📜 Notes

Divs are generic containers used to group elements. Prefer semantic elements when possible.

📖 Classes and IDs

HTML CSS Selectors Structure

🕯️ Incantation


    <div class="spell-card" id="html">

    </div>

📜 Notes

  • Classes can be reused on multiple elements.
  • IDs should only be used once per page.
  • Classes are selected with .
  • IDs are selected with #
