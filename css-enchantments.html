🔥 Display Flex

CSS Flexbox 
                                 
display:flex;

📜 Notes

  • First line to use Flexbox.
  • Turns element into a flex container.
  • Children become flex items.
  • Enables alignment and spacing.
  • Parent gets display:flex;, not the children.
