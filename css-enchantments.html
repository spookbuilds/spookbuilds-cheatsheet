🔥 Display Flex

CSS Layout Flexbox Containers 
                                 
.container {

    display: flex;

}

📜 Notes

  • Turns the selected element into a flex container.
  • Direct children become flex items.
  • Used for arranging elements horizontally or vertically.
  • Applied to the parent element.

🔗 Often Used With

  • justify-content
  • align-items
  • gap
  • flex-direction

📅 Added: 17 July 2026

🔥 Justify Content

CSS Layout Flexbox Alignment 
                                 
.container {

    display:flex;

    justify-content:center;

}

📜 Notes

  • Controls alignment on the main axis.
  • Default flex direction is row.
  • In a row:
    horizontal = justify
    vertical = align
  • Common values:
    justify-content:flex-start;
    justify-content:center;
    justify-content:flex-end;
    justify-content:space-between;
    justify-content:space-around;
    justify-content:space-evenly;

🔗 Often Used With

  • display:flex
  • flex-direction
  • gap

📅 Added: 17 July 2026

📦 Width

CSS Sizing Box Model Layout 
                                 
.card {

    width: 300px;

}

📜 Notes

  • Controls how wide an element is.
  • Usually applied to containers, cards, images, sections.
  • 300pxmeans the element is 300 pixels wide.
  • Width only affects the content area by default.
  • Padding and borders can make the final size larger unless usingbox-sizing.

🔗 Often Used With

  • max-width
  • height
  • box-sizing
  • margin

📦 Margin

CSS Spacing Box Model 
                                 
.card {

    margin: 20px;

}

📜 Notes

  • Creates space OUTSIDE an element.
  • Pushes other elements away.
  • Does not affect the element's background.
  • Common versions:
    margin-top:20px;
    margin-right:10px;
    margin-bottom:30px;
    margin-left:5px;
  • Shortcut: margin:10px 20px;=
    top/bottom = 10px
    left/right = 20px

🔗 Often Used With

  • padding
  • gap
  • width
  • flexbox
