🪄 Initialising a Repository

                                 
git init

📜 Notes

  • Creates a new Git repository in the current folder.

📅 Added: 17 July 2026

📦 Stage Everything

                                 
git add .

📜 Notes

  • Stages all changed files ready for a commit.

📅 Added: 17 July 2026

📝 Create a Commit

                                 
git commit

📜 Notes

  • Creates a snapshot of your project.
  • Opens in VS Code to edit commit message.

📅 Added: 17 July 2026

🚀 Push to GitHub

                                 
git push origin main

📜 Notes

  • Uploads commits to GitHub.
  • Can take a few minutes to update.

📅 Added: 17 July 2026

🧬 Clone Repository

                                 
git clone git@github.com:spookbuilds/project-name.git

📜 Notes

  • Downloads a GitHub repository to your computer.
  • Create it on GitHub first.

📅 Added: 17 July 2026
